TWStockInfo
===========

[TWStockInfo](https://shengyu7697.github.io/TWStockInfo/) 是一個網站工具, 集合了台股各種大型資訊網站連結.  
使用者可以輸入股票代號, 一次列出所有相關的資訊頁面連結.  
可以快速地打開分頁進行投資分析.  

## 使用說明
輸入股票代號, 例如 2330  
按下 "完成" 按鈕, 即可顯示各種大型網站連結.  
在超連結上按 "滑鼠中鍵" 可快速地另開啟新分頁.  

## 特色
* 支援多個大型網站連結, 目前支援網站有：
    + Yahoo 股市
    + GoodInfo
    + CMoney
    + 鉅亨網
  
* 支援常用的大型股
